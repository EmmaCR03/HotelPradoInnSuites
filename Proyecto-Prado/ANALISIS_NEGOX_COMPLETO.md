## ⚠️ Cosas a Verificar ANTES de Pagar

### Preguntas Importantes:

1. **¿Incluyen backup automático?**
   - Muy importante para un sistema de hotel
   - ¿Es diario, semanal?
   - ¿Cuánto tiempo guardan los backups?

2. **¿Cuál es el límite de tamaño de base de datos?**
   - Plan Avanzado dice "2 bases de datos"
   - ¿Hay límite de tamaño por base de datos?
   - ¿Puedes aumentar el tamaño después?

3. **¿Ofrecen migración gratuita?**
   - Si ya tienes el sitio en otro hosting
   - ¿Te ayudan a migrar?

4. **¿Qué versión de SQL Server incluyen?**
   - ¿SQL Server Express o completo?
   - ¿Qué versión (2019, 2022)?

5. **¿Hay límite de usuarios simultáneos?**
   - Importante para el sistema de reservas
   - ¿Cuántos usuarios pueden estar conectados al mismo tiempo?

6. **¿Puedes escalar recursos después?**
   - Si necesitas más espacio o transferencia
   - ¿Puedes agregar recursos sin cambiar de plan?

7. **¿Política de reembolso?**
   - Por si acaso no te funciona
   - ¿Tienen período de prueba?

8. **¿Soporte técnico incluido?**
   - ¿Es realmente 24/7?
   - ¿Por qué medios (chat, email, teléfono)?
   - ¿Tiempo de respuesta promedio?
