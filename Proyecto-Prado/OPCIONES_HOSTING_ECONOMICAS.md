## ⚠️ Cosas a Verificar Antes de Contratar

### Preguntas Importantes:

1. **¿Incluye SQL Server?**
   - Tu proyecto necesita SQL Server, no MySQL
   - Verifica que el plan incluya SQL Server

2. **¿Qué versión de .NET Framework soportan?**
   - Necesitas .NET Framework 4.8.1
   - La mayoría de proveedores lo soportan, pero verifica

3. **¿Incluyen SSL gratis?**
   - Necesitas HTTPS para producción
   - La mayoría incluyen Let's Encrypt gratis

4. **¿Cuánto espacio incluye la base de datos?**
   - Verifica el límite de tamaño de base de datos
   - Puede ser diferente al espacio en disco

5. **¿Ofrecen backup automático?**
   - Muy importante para un sistema de hotel
   - Algunos lo incluyen, otros es adicional

6. **¿Puedo escalar después?**
   - Verifica que puedas cambiar de plan fácilmente
   - O agregar recursos adicionales
